# E-Commerce Product Management System (TypeScript SBA)

This is my project for the TypeScript and Advanced JavaScript SBA. I built a small product system that calls a public API and shows basic info about each product.

### What I Used

- TypeScript
- Async/await
- Classes
- Simple error handling
- Fetch API

### Features

- Fetch product data from an API
- Show product details
- Calculate discounts
- Calculate taxes
- Use classes for product objects

### How to Run
npm install npm run build npm start
